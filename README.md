# skills

[![skills.sh](https://skills.sh/b/cystema/skills)](https://skills.sh/cystema/skills)

Diagnostic Claude Code skills. Each produces a ranked report — no edits.

## Skills

- **[blast-radius](skills/blast-radius)** — Map what breaks if a change ships wrong. Direct callers, hidden coupling, irreversibility, recovery cost.
- **[error-budget](skills/error-budget)** — Audit error paths. Flag swallowed errors, catches far from source, defensive noise drowning the happy path.
- **[naming-audit](skills/naming-audit)** — Audit names as one concern. Flag drift, lies, under-description across files/modules.
- **[spark-joy](skills/spark-joy)** — KonMari review. Keep only what earns its place; flag shallow, untestable, low-glance-value code.
